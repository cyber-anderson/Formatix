# Formatix Image Converter

Formatix is a fast, modern, and user-friendly batch image converter built with Python. It is designed for those who need to quickly convert image formats, resize files, or generate high-quality `.ico` files without any unnecessary complexity.

## Key Features

* **Batch Processing:** Process dozens or hundreds of images simultaneously.
* **Drag-and-Drop:** Simply drag and drop your files directly into the application window.
* **ICO Generation:** Smart creation of multi-layered icons for Windows.
* **Modern UI:** Clean dark-themed interface with High DPI support.
* **High Performance:** Multi-threaded processing ensures the interface remains responsive while you work.

## Requirements

* **Python 3.x**
* **Pillow** (the core library for image processing)

2. Install the required dependencies:
   ```bash
   pip install pillow

(Note: If you plan to use the drag-and-drop feature, ensure you have tkinterdnd2 installed as well).

    Run the application:
    Bash

    python Formatix_Image_Converter_v1.8.2.py

How to Use

    Launch the application.
    Drag and drop your images into the app or click the "Add Files" button.
    Choose your desired output format, target size, and other settings.
    Click the "Convert" button and let Formatix handle the rest!

Tech Stack

    Language: Python
    GUI: Tkinter
    Image Processing: Pillow (PIL)
